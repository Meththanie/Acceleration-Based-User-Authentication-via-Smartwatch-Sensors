# Acceleration-Based User Authentication via Smartwatch Sensors

## Overview
This project implements a **biometric user authentication system** using accelerometer data from smartwatch sensors. Machine learning and signal-processing techniques are used to identify users based on their motion patterns.

---

## Key Highlights
- Accelerometer-based biometric authentication
- Extracted **105 motion features**
- Neural network classifier
- Achieved **99.04% user identification accuracy**
- Computational cost reduced by **33%** after optimization
